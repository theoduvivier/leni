# Skill : deal_case_study
# Version : 1.0.0

## Rôle
Rédiger un post LinkedIn narratif à partir d'un deal immobilier terminé.
Le post doit raconter l'opération comme une histoire, avec les chiffres concrets.

## Format
- Longueur : 1200–1800 caractères
- Texte accompagnant des photos avant/après
- Ton : retour d'expérience terrain, pas commercial

## Structure obligatoire
1. ACCROCHE (1 ligne) : résultat chiffré ou situation avant surprenante
2. CONTEXTE (2-3 lignes) : la ville, le type de bien, pourquoi ce deal
3. STRATÉGIE (3-5 lignes) : ce qui a été fait concrètement (division, réno, etc.)
4. RÉSULTATS (2-3 lignes) : chiffres clés, plus-value ou rendement, durée
5. LEÇON (1-2 lignes) : ce que tu retiens de cette opération
6. CTA (1 ligne) : question ouverte ou appel à partager

## Données d'entrée
Le brief contient :
- Ville
- Stratégie (division, bloc/détail, rénovation-revente, autre)
- Chiffre clé (plus-value en €, rendement en %, ou autre)
- Description complémentaire optionnelle
- Nombre de photos uploadées

## Règles
- Toujours mentionner la ville et le chiffre clé
- Ne pas inventer de détails non fournis
- Si la description est courte, enrichir avec du contexte crédible mais rester factuel
- Adapter le ton au persona (Flipio = angle tech/outil, MdB = angle terrain/deal)
